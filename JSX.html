O que é JSX no React?

JSX (JavaScript XML) é uma extensão de sintaxe para JavaScript usada no React. Ele permite que você escreva elementos da interface de usuário de forma semelhante ao HTML, mas com o poder do JavaScript embutido.

Exemplo 1: Renderizando um elemento simples

import React from "react";

function App() {
  return (
    <div>
      <h1>Olá, mundo!</h1>
      <p>Bem-vindo ao mundo do React com JSX.</p>
    </div>
  );
}

export default App;

Exemplo 2: Expressões JavaScript em JSX

JSX permite incluir expressões JavaScript diretamente, usando chaves {}.

const nome = "João";
const idade = 25;

function Saudacao() {
  return (
    <div>
      <h1>Olá, {nome}!</h1>
      <p>Você tem {idade} anos.</p>
    </div>
  );
}

Características do JSX

  • Expressões JavaScript: Use qualquer expressão com {}.
  • Atributos: Use className em vez de class.
  • Fragmentos: Use <React.Fragment> ou <>....
