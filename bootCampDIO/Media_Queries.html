Guia de Pseudo-classes e Media Queries no React

1. Pseudo-classes no React

Pseudo-classes permitem estilizar elementos com base em seus estados (hover, focus, active). No React, isso pode ser feito via CSS ou styled-components.

Exemplo com CSS

Crie um arquivo App.css:

.botao { background-color: blue; color: white; padding: 10px; border: none; cursor: pointer; }
.botao:hover { background-color: darkblue; }
.botao:active { background-color: red; }

E use no componente React:

import "./App.css";
function App() {
return <button className="botao">Clique aqui</button>;
} export default App;

Exemplo com Styled-components

import styled from "styled-components";
const Botao = styled.button`
background-color: blue; color: white; padding: 10px; border: none; cursor: pointer;
&:hover { background-color: darkblue; }
&:active { background-color: red; }
`;
function App() { return <Botao>Clique aqui</Botao>; }
export default App;

2. Media Queries no React

Media queries ajustam o layout conforme o tamanho da tela. No React, podemos usar CSS ou styled-components.

Exemplo com CSS

Arquivo App.css:

.conteiner { width: 80%; margin: auto; padding: 20px; background-color: lightgray; }
@media (max-width: 600px) {
.conteiner { width: 100%; background-color: lightblue; }
}

Exemplo com Styled-components

import styled from "styled-components";
const Conteiner = styled.div`
width: 80%; margin: auto; padding: 20px; background-color: lightgray;
@media (max-width: 600px) { width: 100%; background-color: lightblue; }
`;
function App() { return <Conteiner>Texto responsivo</Conteiner>; }
export default App;

Exemplo com Hooks

Para alterar o estado com base na largura da tela:

import { useState, useEffect } from "react";
function App() {
const [telaPequena, setTelaPequena] = useState(window.innerWidth < 600);
useEffect(() => {
const handleResize = () => setTelaPequena(window.innerWidth < 600);
window.addEventListener("resize", handleResize);
return () => window.removeEventListener("resize", handleResize);
}, []);
return <div style={{ backgroundColor: telaPequena ? "lightblue" : "lightgray", padding: "20px" }}>
{telaPequena ? "Tela Pequena" : "Tela Grande"}
</div>;
}
export default App;

Conclusão

Com esses exemplos, você pode utilizar pseudo-classes e media queries para tornar seus componentes React mais interativos e responsivos.
