Manual Interativo de JavaScript

JavaScript é uma das linguagens de programação mais populares para desenvolvimento web. Este manual explora conceitos desde o básico até o avançado com exemplos práticos.

Documentação oficial: MDN JavaScript

1. Operações Básicas

Operações matemáticas em JavaScript são simples e intuitivas.

// Soma console.log(5 + 3); // 8 // Subtração console.log(10 - 6); // 4 // Multiplicação console.log(4 * 2); // 8 // Divisão console.log(8 / 2); // 4

Documentação oficial: MDN Operadores Aritméticos

2. Condicionais

Usamos condicionais para executar código com base em uma comparação.

let idade = 18; if (idade >= 18) { console.log("Você é maior de idade."); } else { console.log("Você é menor de idade."); }

Documentação oficial: MDN Condicionais

3. Tipos de Variáveis

Em JavaScript, há três formas principais de declarar variáveis.

var nome = "Alice"; // Variável global let idade = 25; // Variável de escopo limitado const PI = 3.14; // Constante imutável

Documentação oficial: MDN Variáveis

4. Laços de Repetição

Usamos loops para repetir um bloco de código.

// Loop for for (let i = 0; i < 5; i++) { console.log("Número: " + i); } // Loop while let contador = 0; while (contador < 3) { console.log("Contador: " + contador); contador++; }

Documentação oficial: MDN Laços de Repetição

5. Arrow Functions

Arrow functions são uma forma simplificada de escrever funções em JavaScript.

const somar = (a, b) => a + b; console.log(somar(5, 3)); // 8

Documentação oficial: MDN Arrow Functions

6. Arrays

Arrays armazenam múltiplos valores em uma única variável.

let frutas = ["Maçã", "Banana", "Laranja"]; console.log(frutas[1]); // Banana

Documentação oficial: MDN Arrays

7. Funcionalidades ES6

O ES6 trouxe diversas melhorias para JavaScript.

// Template Literals let nome = "Alice"; console.log(`Olá, ${nome}!`); // Destructuring let pessoa = { nome: "João", idade: 30 }; let { nome, idade } = pessoa; console.log(nome, idade);
